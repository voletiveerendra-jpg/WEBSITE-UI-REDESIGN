This project is a Starbucks Landing Page UI Design created in Figma. The design includes a navigation bar, promotional offer banner, and a hero section showcasing Starbucks beverages. The objective of this project is to practice modern web UI/UX design principles and improve Figma design skills.
🎯 Features
Responsive Landing Page Layout
Starbucks Branding & Logo
Navigation Menu (Home, Shop All, About)
Search Functionality Icon
Login and Cart Icons
Promotional Offer Banner
High-Quality Hero Image
Clean and Modern User Interface
🛠️ Tools Used
Figma – UI Design
Google Fonts – Typography
Starbucks Assets – Logo & Product Images
Icon Library – Search, Login, Cart Icons
